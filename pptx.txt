HOW DOES KRATER WORK?
Find and Upload
The citizens snap a picture of the pothole and report it via KRATER.
AI Analysis
The system scans the image, drafting a report with severity and budget estimates. Then the user clicks "Report".
Admin Verification
The Admin receives the ticket, verifies the data, and chooses to "Confirm” or “Decline”.
Action
The Admin assigns a Road Inspector to physically verify the AI's report and then earmarks the preliminary funds. The citizen receives real-time tracking updates throughout the process.
